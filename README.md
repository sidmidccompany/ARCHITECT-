# ARCHITECT

This repository contains the **AVF Creative Architect ERP** Odoo module.

The module provides a basic framework for managing architectural projects in Odoo. It includes menu items, models, access rules and placeholder assets as a starting point for further development.

## Structure

- `avf_architect/` – main module directory
    - `__manifest__.py` – module manifest
    - `models/` – Python models
    - `views/` – basic menu and views
    - `security/` – access control definitions
    - `static/` – placeholder CSS and JavaScript files
    - `data/` and `demo/` – initial and demo data

To install the module, add this repository to your Odoo addons path and update the app list.
